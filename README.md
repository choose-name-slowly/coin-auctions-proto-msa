## 🙌 Coin Auction
코인으로 거래하는 실시간 경매 서비스

## ❓프로젝트 소개
- 다양한 중고 물품을 경매에 등록하고 입찰할 수 있는 플랫폼으로, 입찰과 결제는 암호화폐(코인)로 이루어집니다.
- 자원을 재활용하는 동시에 암호화폐의 새로운 거래처를 제공합니다.
- 이를 통해 환경 보호와 함께 암호화폐의 실용성을 높이는 긍정적인 효과를 기대해봅니다.

- 🔐 **경매**: 
- 🗂️ **입찰**: 
- 🗂️ **코인 및 결재**: 
- 🏪 **채팅**: 경매의 라이프사이클에 따른 채팅방의 생성과 종료
